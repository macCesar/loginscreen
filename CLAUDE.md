This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

Titanium Alloy mobile authentication app (iOS/Android/iPad) using SDK 13.1.1.GA. Three-screen flow: Login → Sign Up → Forgot Password.

## Build Commands

```bash
ti build -p ios -T simulator      # Build/run iOS Simulator
ti build -p android -T emulator   # Build/run Android emulator
alloy compile                      # Compile Alloy views to Resources/ (no run)
```

No automated tests or linting configured.

## Architecture

**MVC Structure (Alloy Framework):**
- `app/controllers/` - Screen logic (JS)
- `app/views/` - XML view definitions
- `app/styles/` - TSS styles (scoped by view ID/class)
- `app/lib/helpers/` - Shared utilities (`validation.js`, `animation.js`)
- `app/assets/` - Images, fonts, `semantic.colors.json`
- `tiapp.xml` - App manifest, SDK version, platform settings

**NEVER edit:** `Resources/` or `build/` (auto-generated by Alloy compiler)

**File naming:** Controller, view, and style files share base names (e.g., `index.js`, `index.xml`, `index.tss`)

## Shared Helpers

**`lib/helpers/validation.js`** - Email regex validation, field error show/hide with animations, underline color updates

**`lib/helpers/animation.js`** - Shake animation, button touch feedback, event listener cleanup management

## Event Cleanup Pattern

Controllers track listeners in `_listeners` array, provide `cleanup()` function, and bind to window close:
```javascript
$.win.addEventListener('close', cleanup)
```
Modal dialogs call `ctrl.cleanup()` on close.

## Design System (app.tss)

Colors:
- `#1a1a2e` (bg-dark), `#16213e` (bg-card), `#0f3460` (bg-accent)
- `#e94560` (primary), `#c81d4e` (primary-dk)
- `#ffffff` (text-main), `#8d99ae` (text-muted)
- `#e63946` (error), `#4ecca3` (success)

Shared classes: `.bgTop`, `.bgBottom`, `.header`, `.backBtn`, `.fieldGroup`, `.field`, `.fieldLabel`, `.fieldError`, `.primaryBtn`, `.primaryBtnLabel`, `.screenIcon`, `.screenTitle`, `.screenSubtitle`

## Coding Style

- `const`/`let` (no `var`), single quotes, **no semicolons**
- Indentation: tabs
- XML event handlers use string references: `onClick="doLogin"`
- TSS selectors: `#id` and `.class` tied to view XML
